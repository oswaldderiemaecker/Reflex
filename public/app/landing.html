¿Qué hay de nuevo?

Reflex 360º contiene una serie de herramientas que llevara el control de la visita medica a otro nivel:

  • Cliente Android, aplicación 100% nativa, gran rendimiento y performance.
  • Cliente Web, si no tienes telefonos puedes ingresar por la web.
  • Visitas y Cobertura en tiempo real.
  • Geo-localizamos tu target médico.
  • Programar medicos y puntos de contactos nunca fue tan facil.

Soporte

The quickest way to get help is to ask @PostgresApp on Twitter, or to open an issue on Github.

When reporting bugs, let us know which version of Postgres.app & OS X you are using, and be sure to include detailed error messages, even if your issue seems similar to another one.

Quieres una demo?

Want to contribute to the project? Here are some great ways you can help:

  • Troubleshoot reported issues
  • Answer questions on Stack Overflow
  • Test pre-release versions of the app
  • Fork Postgres.app, fix a bug, and send a pull request
  • Improve the documentation

Para mayor información o como colaborar en el proyecto, enviar un correo a David Tataje: davidtataje@gmail.com.
